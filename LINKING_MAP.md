## Step-by-Step for Each Market

### KSA (Saudi Arabia)
1. Open Excel, go to "2026 Sufficiency" sheet
2. Select cells **B5:AH11** (includes all KSA brands)
3. Copy (`Ctrl+C`)
4. Go to PPT slide **15 or 16**
5. Paste Special > Paste Link

### GNE (Gulf North East)
1. Select cells **B13:AH18**
2. Link to PPT slide **22 or 23**

### South Africa
1. Select cells **B20:AH27**
2. Link to PPT slide **30 or 31**

### Turkey
1. Select cells **B29:AH34**
2. Link to PPT slide **38 or 39**

### Pakistan
1. Select cells **B36:AH40**
2. Link to PPT slide **46 or 47**

### Egypt
1. Select cells **B42:AH44**
2. Link to PPT slide **54 or 55**

### Morocco
1. Select cells **B46:AH46**
2. Link to PPT slide **62 or 63**

### FSA (French Speaking Africa)
1. Select cells **B48:AH50**
2. Link to PPT slide **70 or 71**

### Kenya
1. Select cells **B52:AH54**

### Nigeria
1. Select cells **B58:AH59**
